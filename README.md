# Song Vibe Matcher

Song Vibe Matcher is an intelligent music analysis tool that dives deep into the underlying themes, subtext, and sentiment of your favorite tracks to generate unique AI-powered visuals and captions.

## The Core Experience

The heart of the application follows a seamless flow:
**🎧 Input: Song** → **🧠 Analysis: Sentiment + Theme + “Vibe”** → **🎨 Output: Generated Cover Image + Caption**

## Features

- **Flexible Input**: Select songs from YouTube, SoundCloud, Last.fm, or other popular platforms.
- **Deep Analysis**: Uses advanced AI to decode the emotional core and underlying message of any track.
- **Visual & Textual Output**: Automatically generates a unique, sentiment-based cover image and a meaningful caption that bridges the artist's intent with the user's taste.
- **Incremental Playlists (Optional)**: While the focus is on standalone song analysis, you can also build incremental, indefinite playlists that evolve with your vibe.

## Technology Stack

- **Frontend**: Angular 21 (Modern, High-Interaction UI)
- **Backend**: FastAPI (Python 3)
- **Database**: MongoDB for user data & Vector Database for song embeddings
- **Generative AI**: Open-source models for high-quality image and text generation

## Roadmap

Building this project in tangible, incremental steps:
1. **v0.1**: Local text sentiment analysis for individual songs. (in progress...)
2. **v0.2**: Integration of AI Image Generation.
